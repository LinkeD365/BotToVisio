### ![smallIcon 32](https://user-images.githubusercontent.com/43988771/146911586-792ac88e-5e9e-45a9-9936-aa8bcb8888db.png)  PVA to Visio

[Power Virtual Agents](https://powervirtualagents.microsoft.com/en-us/) are Microsoft's chat bot tool to allow interaction with your customers and employees in a native way. 

This tool allows documentation of the Topics associated with a Chatbot.

Full walkthrough can be found on my blog [here](https://linkeD365.blog)

1. Connect to your Dataverse environment
2. Select the tool in the list
3. Choose your bot 
4. Choose one or more topics
5. Hit the Visio button


![Snag_5e42cc](https://user-images.githubusercontent.com/43988771/146912488-62fa5f61-d75f-4adf-ac12-5efd4fea35f6.png)

6. Open the Visio if you want
